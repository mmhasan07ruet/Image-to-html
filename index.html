Introducing Pico

A stupidly simple & blazing fast, flat file CMS.

Making the web easy.

Stupidly Simple

Picos makes creating and maintaining a website as simple as editing text files.

Blazing Fast

Pico is seriously lightweight and doesn't use a database, making it superfast.

No Database

Pico is a "flat file" CMS, meaning no database woe's, no MySQL queris, nothing.

Markdown Formatting

Edit your website in your favourite text editor using simple Markdown formatting.

Twig Templates

Pico uses the Twig templating engine, for powerful and flexible themes.

Open Source

Pico is completely free and open source, released under the MIT license.

Want to Contribute?

Help make Pico better by checking out the Github repository and submitting pull requests.

If you find a bug please report it on the issues page.
